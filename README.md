Lottery of Birth: If you were born here..
"You didn't choose where you were born. But where you were born chose almost everything about you."

This is a visualisation project where showcase global inequality in a more personalised way, rather than big numbers and excel sheets. Gloabl Inequality is a known fact but we sometimes undermine how big the gap actually is. For example on an everage a high income country's GNI per capita(PPP adjusted) is atleast 40k more than those of low income country's.

We selected 20 countries, 5 countries for each category:
High Income Category: United States, Australia, Canada, Japan and Germany
Upper Middle Income: China, Brazil, Mexico, Turkey, South Africa
Lower Middle Income: India, Vietnam, Egypt, Zimbabwe, Bangladesh
Low Income: Ethiopia, Afghanistan, Niger, Chad, Somalia

To run this project, at the Lottery of Birth Folder, run following commands to make sure dependencies are fulfilled
 ```bash
npm install
npm install lucide-react
npm run dev
```
Point to remember, once you finish a chapter you can't go back to the previous chapter, just like how life is, you can't go back to the time that has already passed.
And you're good to go! Feel free to ignore the other folders, those were our graph prototype implementations and data processing scripts.
